数字信号处理--随机变量与随机信号
发表于 2020-03-31| 更新于 2020-04-01
字数总计: 3.3k|阅读时长: 16 分钟|阅读量:

重读《数字信号处理》，主要想对下册统计数字信号部分进行梳理。因为人发出的语音信号在每次观察下都是随机的，但是有别与一般的随机变量，它加入了时间这个维度。故针对一个随机信号有两个随机因素，时间ttt和样本观察nnn。先好好的分析一波随机变量，再加入时间推广到随机信号。

随机变量

随机变量XXX：描述自然件随机事件

  • 连续值：服从如正态分布等
  • 离散值：服从如二项分布、泊松分布

表达以上分布的方式：分布函数/概率密度/数字特征

均值、均方值、方差、矩

这几个概念讲解的顺序为：

  1. 均值
  2. 均方值、方差
  3. 矩

⭕️均值：

μX=E{X}=∫−∞∞xp(x)dx\mu_{X}=E\{X\}=\int_{-\infty}^{\infty} x p(x) \mathrm{d} x μX​=E{X}=∫−∞∞​xp(x)dx

⭕️均方值、方差：

DX2=E{∣X∣z}=∫−∞∞∣x∣2p(x)dxσX2=E{∣X−μX∣2}=∫−∞∞∣x−μX∣2p(x)dx\begin{array}{c}{D_{X}^{2}=E\left\{|X|^{z}\right\}=\int_{-\infty}^{\infty}|x|^{2} p(x) \mathrm{d} x} \\ {\sigma_{X}^{2}=E\left\{\left|X-\mu_{X}\right|^{2}\right\}=\int_{-\infty}^{\infty}\left|x-\mu_{X}\right|^{2} p(x) \mathrm{d} x}\end{array} DX2​=E{∣X∣z}=∫−∞∞​∣x∣2p(x)dxσX2​=E{∣X−μX​∣2}=∫−∞∞​∣x−μX​∣2p(x)dx​

这里可以发现，这两个概念都可以用E{⋅}E\{\cdot\}E{⋅}均值去解释。均方值是对所有样本【所有随机事件】的平方求均值，方差是对离中心点的平方求均值。

⭕️引出高阶矩的概念：

定义原点矩：

ηXn=E{∣X∣m}=∫∞∞∣x∣mp(x)dx\eta_{X}^{n}=E\left\{|X|^{m}\right\}=\int_{\infty}^{\infty}|x|^{m} p(x) \mathrm{d} x ηXn​=E{∣X∣m}=∫∞∞​∣x∣mp(x)dx

定义中心矩：

γX=E{∣X−μX∣m}=∫−∞∞∣x−μX∣mp(x)dx\gamma_{X}=E\left\{\left|X-\mu_{X}\right|^{m}\right\}=\int_{-\infty}^{\infty}\left|x-\mu_{X}\right|^{m}{p}(x) \mathrm{d} x γX​=E{∣X−μX​∣m}=∫−∞∞​∣x−μX​∣mp(x)dx

可以很轻易的发现均方值和方差是引入矩定义后m=2m=2m=2下的情况。

根据m的大小可以定义，一阶、二阶、三阶、四阶统计量，等

三阶统计量：斜度

 Skew =E{[X−μXσX]3}=1σX3γX3\text { Skew }=E\left\{\left[\frac{X-\mu_{X}}{\sigma_{X}}\right]^{3}\right\}=\frac{1}{\sigma_{X}^{3}} \gamma_{X}^{3}  Skew =E{[σX​X−μX​​]3}=σX3​1​γX3​

四阶统计量：峰度

 Kurtosis =E{[X−μxσx]4}−3=1σx4γx4−3\text { Kurtosis }=E\left\{\left[\frac{X-\mu_{x}}{\sigma_{x}}\right]^{4}\right\}-3=\frac{1}{\sigma_{x}^{4}} \gamma_{x}^{4}-3  Kurtosis =E{[σx​X−μx​​]4}−3=σx4​1​γx4​−3

上帝视角与样本的关系

因为之前在机器学习中，对一些概念性的理解产生了点误解，频率派和贝叶斯派的区分中，有涉及到上帝视角。

核心在于：p(x)p(x)p(x)的求解

  • 若来源于样本的且为离散变量，则可以直接用频率来代替概率计算该值【大数定理】
  • 若来源于假设，离散如二项式分布，连续如正太分布，其各自拥有各自的分布律函数。则p(x)p(x)p(x)与样本无关，我称之为上帝视角或者理想真值，因为当假设分布确定时，则p(x)p(x)p(x)就已经确定了。

多维随机变量

当随机变量XXX拥有NNN维特征：

X=[X1,X2,…,XN]TX = [X_1,X_2,\dots,X_N]^T X=[X1​,X2​,…,XN​]T

则均值为：

μX=[μX1,μX2,⋯ ,μXN]T,μXi=E{Xi}\boldsymbol{\mu}_{X}=\left[\mu_{X_{1}}, \mu_{X_{2}}, \cdots, \mu_{X_{N}}\right]^{\mathrm{T}}, \mu_{X_{i}}=E\left\{X_{i}\right\} μX​=[μX1​​,μX2​​,⋯,μXN​​]T,μXi​​=E{Xi​}

在这里统一用XXX代表随机变量，用xxx代表单个样本

则方差矩阵为：

Σ=E{(X−μX)⋅(X−μX)T}=[σ12cov(X1,X2)...cov(X1,XN)cov(X2,X1)σ22...cov(X2,XN)⋮⋮⋮⋮cov(XN,X1)cov(XN,X2)⋯σN2]\begin{aligned} \boldsymbol{\Sigma}= & E\left\{\left(\boldsymbol{X}-\boldsymbol{\mu}_{X}\right) \cdot\left(\boldsymbol{X}-\boldsymbol{\mu}_{X}\right)^{\mathrm{T}}\right\}\\ =& \left[ \begin{matrix} \sigma _{1}^{2}& cov\left( X_1,X_2 \right)& ...& cov\left( X_1,X_N \right)\\ cov\left( X_2,X_1 \right)& \sigma _{2}^{2}& ...& cov\left( X_2,X_N \right)\\ \vdots& \vdots& \vdots& \vdots\\ cov\left( X_N,X_1 \right)& cov\left( X_N,X_2 \right)& \cdots& \sigma _{N}^{2}\\ \end{matrix} \right] \end{aligned} Σ==​E{(X−μX​)⋅(X−μX​)T}⎣⎢⎢⎢⎡​σ12​cov(X2​,X1​)⋮cov(XN​,X1​)​cov(X1​,X2​)σ22​⋮cov(XN​,X2​)​......⋮⋯​cov(X1​,XN​)cov(X2​,XN​)⋮σN2​​⎦⎥⎥⎥⎤​​

不同维度之间的计算用协方差定义的，这里的cov(Xi,Xj)cov(X_i,X_j)cov(Xi​,Xj​)最终得到的是实数。

协方差，不仅不同维度之间可以计算，也可扩展成两个多维随机变量之间的定义，此时结果就不再是实数了，而是协方差矩阵。cov(X,Y)=E{(X−μX)(Y−μY)T}cov(X,Y)=E\{(X-\mu_X)(Y-\mu_Y)^T\}cov(X,Y)=E{(X−μX​)(Y−μY​)T}

随机信号

这里温漂电压是一个随机信号。在相同的条件下独立地进行多次观察。

随机信号x(n,i)x(n,i)x(n,i)有以下四种情况：

  • 若nnn固定（时间固定），iii变动，则为随机变量

  • 若iii固定（单样本），nnn变动，则为一个一维离散时间序列

  • 若iii和nnn都变动，则x(n,i)x(n,i)x(n,i)是一个随机信号。

  • 若iii和nnn都固定，则x(n,i)x(n,i)x(n,i)是一个实数。

对随机变量，即样本iii变动，可做如下统计变量【集总平均】：

  • 均值

    μX(n)=E{X(n)}=lim⁡N→∞1N∑i=1Nx(n,i)\mu_{X}(n)=E\{X(n)\}=\lim _{N \rightarrow \infty} \frac{1}{N} \sum_{i=1}^{N} x(n, i) μX​(n)=E{X(n)}=N→∞lim​N1​i=1∑N​x(n,i)

  • 方差

    σx2(n)=E{∣X(n)−μX(n)∣2}=lim⁡N→∞1N∑i=1N∣x(n,i)−μX(n)∣2\sigma_{x}^{2}(n)=E\left\{\left|X(n)-\mu_{X}(n)\right|^{2}\right\}=\lim _{N \rightarrow \infty} \frac{1}{N} \sum_{i=1}^{N}\left|x(n, i)-\mu_{X}(n)\right|^{2} σx2​(n)=E{∣X(n)−μX​(n)∣2}=N→∞lim​N1​i=1∑N​∣x(n,i)−μX​(n)∣2

  • 均方

    DX2(n)=E{∣X(n)∣2}=lim⁡N→∞1N∑i=1N∣x(n,i)∣2D_{X}^{2}(n)=E\left\{|X(n)|^{2}\right\}=\lim _{N \rightarrow \infty} \frac{1}{N} \sum_{i=1}^{N}|x(n, i)|^{2} DX2​(n)=E{∣X(n)∣2}=N→∞lim​N1​i=1∑N​∣x(n,i)∣2

  • 自相关函数

    rX(n1,n2)=E{X∗(n1)X(n2)}=lim⁡N→∞1N∑i=1Nx∗(n1,i)x(n2,i)r_{X}\left(n_{1}, n_{2}\right)=E\left\{X^{*}\left(n_{1}\right) X\left(n_{2}\right)\right\}=\lim _{N \rightarrow \infty} \frac{1}{N} \sum_{i=1}^{N} x^{*}\left(n_{1}, i\right) x\left(n_{2}, i\right) rX​(n1​,n2​)=E{X∗(n1​)X(n2​)}=N→∞lim​N1​i=1∑N​x∗(n1​,i)x(n2​,i)

  • 自协方差函数

    cov⁡X(n1,n2)=E{[X(n1)−μX(n1)]⋅[X(n2)−μX(n2)]}=lim⁡N→∞1N∑i=1N[x(n1,i)−μX(n1)]⋅[x(n2,i)−μX(n2)]\begin{aligned} \operatorname{cov}_{X}\left(n_{1}, n_{2}\right) &=E\left\{\left[X\left(n_{1}\right)-\mu_{X}\left(n_{1}\right)\right] \cdot\left[X\left(n_{2}\right)-\mu_{X}\left(n_{2}\right)\right]\right\} \\ &=\lim _{N \rightarrow \infty} \frac{1}{N} \sum_{i=1}^{N}\left[x\left(n_{1}, i\right)-\mu_{X}\left(n_{1}\right)\right] \cdot\left[x\left(n_{2}, i\right)-\mu_{X}\left(n_{2}\right)\right] \end{aligned} covX​(n1​,n2​)​=E{[X(n1​)−μX​(n1​)]⋅[X(n2​)−μX​(n2​)]}=N→∞lim​N1​i=1∑N​[x(n1​,i)−μX​(n1​)]⋅[x(n2​,i)−μX​(n2​)]​

注：以上所有的统计后的量都是关于nnn的变量

    μX(n)=E{X(n)}=μX\mu_X(n)=E\{X(n)\} = \mu_X μX​(n)=E{X(n)}=μX​

  2. 方差也为常数

    σX2(n)=E{∣X(n)−μX∣2}=σX2\sigma^2_X(n) = E\{|X(n)-\mu_X|^2\} = \sigma^2_X σX2​(n)=E{∣X(n)−μX​∣2}=σX2​

  3. 自相关函数仅与选取点有关

    rX(n1,n2)=E{X∗(n)X(n+m)}=rX(n2−n1)r_X(n_1,n_2) = E\{X*(n)X(n+m)\}=r_X(n_2-n_1) rX​(n1​,n2​)=E{X∗(n)X(n+m)}=rX​(n2​−n1​)

平稳随机信号的各态遍历性

rx(m)=E{X(n)X(n+m)}=lim⁡N→∞1N∑i=1Nx(n,i)x(n+m,i)r_{x}(m)=E\{X(n) X(n+m)\}=\lim _{N \rightarrow \infty} \frac{1}{N} \sum_{i=1}^{N} x(n, i) x(n+m, i) rx​(m)=E{X(n)X(n+m)}=N→∞lim​N1​i=1∑N​x(n,i)x(n+m,i)

但是要精确地求出上值，需要无穷多个样本，这在实际工作中是不可能的。
