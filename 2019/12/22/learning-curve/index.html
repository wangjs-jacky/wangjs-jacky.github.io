learning-curve

字数总计: 549|阅读时长: 1 分钟|阅读量:

语法：

这个函数的作用为：对于不同大小的训练集，确定交叉验证训练和测试的分数。一个交叉验证发生器将整个数据集分割k次，分割成训练集和测试集。不同大小的训练集的子集将会被用来训练评估器并且对于每一个大小的训练子集都会产生一个分数，然后测试集的分数也会计算。然后，对于每一个训练子集，运行k次之后的所有这些分数将会被平均。

属性：

estimator：所使用的分类器

X: 训练向量

大小：array-like, shape (n_samples, n_features)

n_samples是样本的数量，n_features是特征的数量

y: 目标相对于X分类或者回归

大小：array-like, shape (n_samples) or (n_samples, n_features), optional

train_sizes: 训练样本的相对的或绝对的数字，

大小：array-like, shape (n_ticks,), dtype float or int

这些量的样本将会生成learning curve。如果dtype是float，他将会被视为最大数量训练集的一部分（这个由所选择的验证方法所决定）。否则，他将会被视为训练集的绝对尺寸。要注意的是，对于分类而言，样本的大小必须要充分大，达到对于每一个分类都至少包含一个样本的情况。

cv: 确定交叉验证的分离策略

int, cross-validation generator or an iterable, optional

  • None，使用默认的3-fold cross-validation,

  • integer,确定是几折交叉验证

–一个作为交叉验证生成器的对象

–一个被应用于训练/测试分离的迭代器

n_jobs : 整数，(默认值为1)。

windows开多线程需要在"name"==__main__中运行。

verbose : 控制冗余

类型：integer, optional（越高，有越多的信息）

返回值：

train_sizes_abs：用于生成learning curve的训练集的样本数。

大小：array, shape = (n_unique_ticks,), dtype int

由于重复的输入将会被删除，所以ticks可能会少于n_ticks.

train_scores : 在训练集上的分数

array, shape (n_ticks, n_cv_folds)

test_scores : 在测试集上的分数

array, shape (n_ticks, n_cv_folds)
